## Features

- Generate XML sitemaps from any URL
- RSS feed generation support
- Automatic URL crawling with cURL
- SEO-friendly output format
- Customizable crawl depth & filters

## Usage

```php
// Basic sitemap generation
php generate.php --url="https://example.com"
```
